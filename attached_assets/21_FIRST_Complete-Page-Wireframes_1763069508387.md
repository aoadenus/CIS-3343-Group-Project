## Form Validations

- Email format: `^[^\s@]+@[^\s@]+\.[^\s@]+$`
- Password: Non-empty
- Show inline errors below each field
- Disable login button until both fields valid

## Responsive

- **Mobile (320px):** Full width, 16px padding, single column
- **Tablet (768px):** Max-width 500px, centered
- **Desktop:** Max-width 500px, centered, padded container

## Data Source & Calculations

**KPIs (Real-Time):**
- Today's Orders: COUNT(orders WHERE date = TODAY)
- In Progress: COUNT(orders WHERE status IN ['In Baking', 'Decorating'])
- Revenue Today: SUM(orders.total_price WHERE date = TODAY)

**Ready for Pickup:**
- Query: SELECT * FROM orders WHERE status = 'Ready for Pickup' AND pickup_date = TODAY
- Sort by pickup_time ASC
- Limit 3, link to "View All"

**Recent Activity:**
- Query: SELECT * FROM activity_log ORDER BY timestamp DESC LIMIT 5
- Include: Order created, status changed, customer added, payment received

SELECT 
  o.OrderID,
  c.Name AS CustomerName,
  o.OrderedDate,
  o.PickupDate,
  o.PickupTime,
  o.OrderStatus,
  o.TotalPrice
FROM orders o
JOIN customers c ON o.CustomerID = c.CustomerID
WHERE o.OrderedDate >= TODAY - 90
ORDER BY o.OrderedDate DESC
