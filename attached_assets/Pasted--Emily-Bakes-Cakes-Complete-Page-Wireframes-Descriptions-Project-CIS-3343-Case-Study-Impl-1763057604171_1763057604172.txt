## Authentication Flow
1. Visit `/staff-login`
2. Enter email + password
3. Backend validates against EMPLOYEE table
4. Returns JWT token
5. Token stored in localStorage
6. All subsequent requests include token
7. Middleware validates token
8. Route guards check user role
