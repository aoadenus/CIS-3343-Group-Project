### 3. **Product Images Are MISSING** üö®
**Problem**:
- Product cards show no cake images
- Visual appeal is eliminated entirely
- Users can't see what they're buying (huge conversion killer)
- Page feels incomplete and unprofessional

**Technical Fix**:
\`\`\`tsx
// In ProductCard.tsx:
<div className="relative overflow-hidden rounded-t-xl 
  aspect-[4/3] bg-gradient-to-br from-[#F5E6D3] to-[#E8D5C4]">
  {product.image ? (
    <img 
      src={product.image} 
      alt={product.name}
      className="w-full h-full object-cover transition-transform 
        duration-300 hover:scale-110"
      loading="lazy"
    />
  ) : (
    <div className="flex items-center justify-center h-full">
      <svg className="w-20 h-20 text-[#C44569]/30" 
        fill="currentColor" viewBox="0 0 20 20">
        {/* Cake icon placeholder */}
        <path d="M10 2a1 1 0 011 1v1.5a4.5 4.5 0 019 0V6a2 2 0 
          012 2v8a2 2 0 01-2 2H2a2 2 0 01-2-2V8a2 2 0 012-2v-1.5a4.5 
          4.5 0 019 0V3a1 1 0 011-1z"/>
      </svg>
    </div>
  )}
</div>
\`\`\`
**Reasoning**:
- Add image container with fixed aspect ratio (4:3) for consistency
- Fallback placeholder SVG icon for missing images
- Hover zoom effect adds interactivity
- Lazy loading improves page performance

### 4. **Sorting Dropdown Needs Clearer Visual Indication** üö®
**Problem**:
- While functional, the dropdown doesn't look obviously clickable
- No clear chevron icon or visual affordance
- Users may not realize it's interactive

**Technical Fix**:
\`\`\`tsx
<div className="relative inline-block">
  <select
    className="appearance-none bg-white border-2 border-[#C44569] 
      rounded-lg px-4 py-2.5 pr-10 
      text-gray-800 font-medium 
      cursor-pointer 
      hover:bg-[#C44569]/5 
      focus:ring-2 focus:ring-[#C44569]/50 
      transition-all duration-200
      shadow-sm hover:shadow-md"
    aria-label="Sort products"
    value={sortOption}
    onChange={handleSortChange}
  >
    <option>Most Popular</option>
    <option>Highest Rated</option>
    <option>Price: Low to High</option>
    <option>Price: High to Low</option>
  </select>
  {/* Custom chevron icon */}
  <svg className="absolute right-3 top-1/2 -translate-y-1/2 
    w-5 h-5 text-[#C44569] pointer-events-none"
    fill="none" stroke="currentColor" viewBox="0 0 24 24">
    <path strokeLinecap="round" strokeLinejoin="round" 
      strokeWidth={2.5} d="M19 9l-7 7-7-7" />
  </svg>
</div>
\`\`\`
**Reasoning**:
- `appearance-none` removes default browser dropdown styling
- Custom chevron SVG makes it obviously a dropdown
- Border color matches brand (raspberry #C44569)
- Hover state (shadow + bg tint) signals interactivity
- Focus ring improves accessibility
